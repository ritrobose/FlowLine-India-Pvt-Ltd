# Tailwind Compiler Skill

This skill parses and translates standard Webflow CSS selectors into Tailwind CSS equivalents.

## Instructions
1. Run `node scripts/compiler.js <css_file_path>` to generate compiled classes list.
2. It processes styling nodes and outputs Tailwind configurations.
